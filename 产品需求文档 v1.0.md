《猫三国》PRD（产品需求文档 v1.0）
一、产品概述
1.1 产品名称

《猫三国》

1.2 产品定位
类型：单机策略游戏（SLG + RTT）
玩法结构：大地图战略 + 半实时战斗
风格：猫拟人 + 三国题材 + 轻策略深系统
1.3 目标用户
用户类型	特征
策略玩家	喜欢《三国群英传》《全面战争》
轻度玩家	可接受自动战斗
三国爱好者	对武将养成有兴趣
1.4 核心卖点
🐱 猫拟人三国（差异化题材）
⚔️ 技能触发链战斗系统（深度玩法）
🧠 半自动战斗（低操作，高策略）
🏙️ 城池经营 + 扩张
🔁 高复玩性（随机武将 / 技能组合）
二、核心玩法循环（最关键）
资源生产 → 武将获取 → 编队 → 出征 → 战斗 → 占领 → 发展 → 再扩张
三、核心系统设计
3.1 大地图系统（策略核心）
玩法说明
玩家控制势力，在地图上占领城池
城池之间通过“节点路径”连接
军队点对点移动
核心功能
功能	描述
城池系统	资源产出 / 防御
行军系统	A→B路径移动
Fog系统	未探索区域不可见
地图交互	点击城池操作
玩家决策点
攻哪座城？
资源 vs 军事优先？
是否分兵？
