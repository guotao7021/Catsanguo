一、系统定位
🎯 一句话定义

战斗UI = 玩家理解战局 + 做出决策的唯一窗口

🎯 核心目标
✅ 清晰展示战场信息（状态、血量、Buff）
✅ 支持操作（技能、阵形切换）
✅ 提供即时反馈（伤害、提示）
✅ 兼容自动战斗 / 手动操作
🧱 二、整体架构
BattleUISystem
 ├── HUD（主界面）
 ├── UnitUI（单位UI）
 ├── SkillUI（技能栏）
 ├── FormationUI（阵形控制）
 ├── InfoUI（战斗信息）
 ├── ResultUI（结算界面）

⚙️ 五、交互设计（核心）
🎯 玩家操作
1️⃣ 技能释放
点击按钮 → 选目标 → 释放
2️⃣ 阵形切换
点击 → 选择阵法 → 应用
3️⃣ 自动战斗
ON → AI控制
OFF → 玩家控制
🔗 六、与系统联动（关键）
1️⃣ 与战斗系统
战斗事件 → UI刷新
2️⃣ 与Buff系统
Buff变化 → 图标更新
3️⃣ 与技能系统
技能CD → UI倒计时
4️⃣ 与阵形系统
阵形切换 → UI提示
