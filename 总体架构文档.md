⚙️ 三、核心设计2：Command Buffer（商业级关键）
❗这是商业项目的“中枢神经”
数据流：
AI / 玩家操作
        ↓
Command Buffer（收集命令）
        ↓
Simulation执行
        ↓
State变化
        ↓
Event派发（表现层）
🧱 命令类型
public interface ICommand {}

public struct AttackCommand : ICommand
{
    public int attacker;
    public int target;
}

public struct CastSkillCommand : ICommand
{
    public int caster;
    public int skillId;
}

🌐 七、核心设计6：网络同步（商业级必须考虑）
推荐模式：Lockstep（确定性同步）
核心要求：

不用随机数（或固定种子）

 → 同步执行
 → 得到一致结果
