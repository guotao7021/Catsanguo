一、系统定位与目标
🎯 系统定位

内政系统 = 资源生产 + 城市发展 + 战争支撑体系

🎯 核心目标
✅ 支撑战斗系统（兵力 / 资源 / 武将成长）
✅ 提供长期养成（城市发展）
✅ 提供策略深度（资源调配 / 建筑选择）
✅ 支持离线收益（挂机/放置）
🧱 二、系统总架构
CitySystem
 ├── ResourceSystem（资源）
 ├── BuildingSystem（建筑）
 ├── ProductionSystem（生产）
 ├── PopulationSystem（人口）
 ├── TaxSystem（税收）
 ├── PolicySystem（政策/科技）
 ├── EventSystem（随机事件）

🧠 核心机制
1️⃣ 等级成长
等级↑ → 产量↑ → 成本↑
2️⃣ 建筑队列（关键）
同时只能升级1~2个建筑
3️⃣ 前置条件
兵营Lv3 → 需要主城Lv5
⚙️ 五、生产系统（ProductionSystem）
🎯 核心

资源 = 时间函数

计算模型
资源增长 = 基础产量 × (1 + 加成)
🧱 实现
resource.current += productionPerSecond * deltaTime;
🔥 加成来源
