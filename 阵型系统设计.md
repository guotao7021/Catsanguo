# 猫三国 - 阵型系统设计文档

> 最后更新：2026-04-15（根据实际代码同步）

## 一、设计思路

### 阵形三层结构
```
阵形 = 数值加成 + 特殊能力 + 战术效果
```

### 已实现状态
- 9种阵型已定义（BattleFormation枚举）
- 阵型配置表已实现（FormationConfigTable）
- 编队时可选择阵型（CityActionDialog.SelectGeneral）
- 战斗中阵型影响Squad属性和特殊能力

## 二、阵法体系（9种，已实现）

```csharp
public enum BattleFormation
{
    FishScale,      // 鱼鳞阵（防御/分摊）
    Square,         // 方阵（防御/稳定）
    Wedge,          // 锥形阵（突破/穿透）
    LongSnake,      // 长蛇阵（机动/连击）
    CraneWing,      // 鹤翼阵（包抄/夹击）
    EightTrigrams,  // 八卦阵（变化/控制）
    CrescentMoon,   // 偃月阵（集中/叠伤）
    Circle,         // 环形阵（包围/集火）
    Vanguard        // 先锋阵（平衡/默认）
}
```

### 分类

| 类型 | 阵型 |
|------|------|
| 防御型 | 鱼鳞阵、方阵、八卦阵 |
| 进攻型 | 锥形阵、长蛇阵、鹤翼阵 |
| 战术型 | 八卦阵、偃月阵、环形阵 |
| 平衡型 | 先锋阵 |

## 三、核心阵法设计

### 3.1 鱼鳞阵（FishScale）
- **特点**：密集排列、层层推进
- **效果**：前排减伤+40%，全队受伤分摊
- **特殊机制**：单体伤害按比例分摊到全队
```
damage = damage * 0.6f;
sharedDamage = damage * 0.4f / teamSize;
```

### 3.2 锥形阵（Wedge）
- **特点**：集中突破一点
- **效果**：前排伤害+50%，穿透后排
- **特殊机制**：破阵能力，攻击前排溢出伤害打后排

### 3.3 鹤翼阵（CraneWing）
- **特点**：两翼展开包围
- **效果**：侧翼单位伤害+30%
- **特殊机制**：包围加成（被两侧夹击→额外+25%伤害）

### 3.4 长蛇阵（LongSnake）
- **特点**：灵活延展
- **效果**：移动速度+30%，攻击频率提升
- **特殊机制**：连续攻击链

### 3.5 八卦阵（EightTrigrams）
- **特点**：动态变化
- **核心机制**：状态切换阵（防御态↔攻击态↔控制态）

| 状态 | 效果 |
|------|------|
| 防御 | 减伤+30% |
| 攻击 | 伤害+25% |
| 控制 | 概率眩晕 |

### 3.6 偃月阵（CrescentMoon）
- **特点**：半月包围
- **效果**：前排抗伤，中后排集中输出
- **特殊机制**：攻击同一目标→叠加伤害

### 3.7 方阵（Square）
- **效果**：全体防御+20%，阵型稳定（抗击退）

### 3.8 环形阵（Circle）
- **效果**：对中心目标伤害+40%

### 3.9 先锋阵（Vanguard）
- **效果**：平衡型，无特殊加减，默认阵型

## 五、阵型在游戏中的使用

### 编队时选择（CityActionDialog.SelectGeneral阶段）

```csharp
// GeneralDeployCard 中的阵型选择
public string SelectedFormation { get; set; }  // "先锋"/"鱼鳞"等中文名

// 转换为枚举
public BattleFormation GetBattleFormation() => SelectedFormation switch
{
    "先锋" => BattleFormation.Vanguard,
    "鱼鳞" => BattleFormation.FishScale,
    "方阵" => BattleFormation.Square,
    "锥形" => BattleFormation.Wedge,
    "长蛇" => BattleFormation.LongSnake,
    "鹤翼" => BattleFormation.CraneWing,
    "八卦" => BattleFormation.EightTrigrams,
    "偃月" => BattleFormation.CrescentMoon,
    "环形" => BattleFormation.Circle,
    _ => BattleFormation.Vanguard
};
```

### 战斗中应用（BattleScene.CreateSquad）

阵型通过FormationConfigTable获取配置，应用于Squad属性计算：
- AttackModifier → Squad攻击力修正
- DefenseModifier → Squad防御力修正
- SpeedModifier → Squad速度修正
- SpecialAbility → 特殊能力逻辑

## 六、与其他系统融合

### 与军种系统
```
阵形 × 军种 = 战斗结果放大器

例：
骑兵 + 锥形阵 = 爆发极强
枪兵 + 方阵 = 防骑无敌
```

### 与技能系统
- 阵形可触发被动技能
- 阵形激活→自动释放特定效果

### 与Buff系统
- 阵形 = 持续Buff + 条件Buff

## 七、数值平衡设计

### 阵形影响权重

| 系统 | 权重 |
|------|------|
| 属性 | 50% |
| 军种 | 25% |
| 阵形 | 15% |
| 技能 | 10% |

### 原则
- 阵形影响结果（但不决定）
- 避免"只要选对阵型就必赢"

## 八、阵型克制关系（规划）

> 来自战斗与阵容搭配系统v2.0设计，待实现

| 阵型 | 被克制 |
|------|--------|
| 锥形阵 | 方阵（稳定防守克冲锋） |
| 方阵 | 鹤翼阵/环形阵（包抄克固守） |
| 鹤翼阵 | 锥形阵（集中突破克分散） |

### 阵型数值范围建议

| 类型 | 数值范围 |
|------|----------|
| 伤害提升 | 10% ~ 30% |
| 减伤 | 15% ~ 25% |
| 首击加成 | 30% ~ 60% |
| 速度影响 | -20% ~ +20% |

### 阵型强度控制
- 阵型收益 ≈ 15%~25% 总战力
- 阵型改变战斗规则，而非单纯数值加成

阵形系统 = 战术层（比数值更高一层）
