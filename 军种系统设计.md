# 猫三国 - 军种系统设计文档

> 最后更新：2026-04-15（根据需求文档同步）

## 一、军种体系（已实现，10类）

```csharp
public enum UnitType
{
    // 基础
    Infantry,        // 步兵
    Cavalry,         // 骑兵
    Archer,          // 弓兵

    // 反制专精
    Spearman,        // 枪兵（反骑）
    ShieldInfantry,  // 盾兵（抗远程）

    // 输出强化
    Crossbowman,     // 强弩
    LightCavalry,    // 轻骑（机动）
    HeavyCavalry,    // 重骑（冲锋）

    // 特殊
    Siege,           // 攻城器械
    Mage             // 法术单位
}
```

### 设计原则
- 每个兵种都有：强项（2个优势）+ 弱点（1~2个）
- 不存在"全能兵种"
- 克制关系是网络结构而非简单三角

### 关键设计点
- 不再只有"完全克制"
- 分层倍率：1.5~1.6（强克）→ 1.2~1.3（弱克）→ 1.0（中性）→ 0.6~0.8（被克）
- 引入"软克制"：步兵 vs 骑兵 = 1.2（非1.5）

## 六、伤害计算公式（已实现）

```
最终伤害 = 基础攻击 × 属性倍率 × 克制倍率 × 技能倍率 × Buff倍率
```

其中属性倍率来自武将6维属性映射：
- **武力** → 基础攻击力加成（AttackModifier）
- **智力** → 技能系数加成（SkillModifier）
- **统帅** → 士气/带兵影响（间接影响兵力规模）

实现类：`DamageCalculator`

## 八、平衡策略

### 平衡三原则
1. 克制 < 属性（防止"无脑克制"）
2. 每个兵种必须有天敌
3. 不允许全能单位

### 数值区间

| 类型 | 倍率范围 |
|------|----------|
| 强克 | 1.4 ~ 1.6 |
| 弱克 | 1.1 ~ 1.3 |
| 被克 | 0.6 ~ 0.8 |

## 十、阵容搭配系统

### 编队规则
- 最多3武将
- 多兵种混编
- 选择1个阵型

### 阵容结构
```
阵容 = 兵种组合 + 武将组合 + 阵型
```

### 流派示例

| 流派 | 组成 | 特点 |
|------|------|------|
| 骑兵突击流 | 骑兵 + 锥形阵 + 猛将型武将 | 高武力加成，快速击破 |
| 防守反击流 | 盾兵 + 方阵 + 统帅型武将 | 高统帅加持士气，消耗对方 |
| 远程消耗流 | 弓兵 + 鹤翼阵 + 谋士型武将 | 高智力技能爆发，远程压制 |
| 枪盾反骑流 | 枪兵 + 盾兵 + 防御型武将 | 统帅+武力兼备，克制骑兵 |

### 武将定位分类（对齐6维属性）

| 类型 | 核心属性 | 作用 | 适配兵种 | 特技类型 | 示例效果 |
|------|----------|------|----------|----------|----------|
| 统帅型 | 统帅(Command) | 全队加成/大部队指挥 | 步兵/盾兵/方阵系 | 指挥特技 | 鼓舞（士气回复加速）、铁壁（全军防御+10%） |
| 猛将型 | 武力(Strength) | 前排输出/冲锋 | 骑兵/重骑/步兵 | 战斗特技 | 突击（首次冲锋伤害+50%）、无双（连续击杀加攻） |
| 谋士型 | 智力(Intelligence) | 战术控制/技能爆发 | 法师/弓兵/强弩 | 战斗特技 | 神射（远程伤害+20%）、火计（AOE持续伤害） |
| 防御型 | 统帅+武力 | 坦克抗压 | 盾兵/枪兵/步兵 | 指挥特技 | 坚守（减伤+护盾）、反击（受击反伤） |

### 武将6维属性与兵种适配

武将的6维属性不仅决定战斗/内政表现，还影响兵种选择和编队策略：

| 属性 | 对兵种的影响 | 推荐搭配 |
|------|-------------|----------|
| 武力(Strength) | 提升物理攻击力，适合近战输出兵种 | 骑兵、重骑、步兵 |
| 智力(Intelligence) | 提升技能伤害和命中率，适合技能型兵种 | 法师、弓兵、强弩 |
| 统帅(Command) | 提升士气和带兵上限，适合大规模部队 | 步兵、盾兵、枪兵 |
| 速度(Speed) | 影响行动频率，适合机动兵种 | 轻骑、骑兵 |

> 政治(Politics)、经济(Economics)、忠诚度(Loyalty) 不直接影响兵种战斗表现，但政治/经济影响征兵效率和军备产出，间接影响可用兵力规模。详见 [Attribute属性系统设计.md](Attribute属性系统设计.md)。

## 十二、最终总结

军种系统 = 多维克制网络（不是简单三角）

当前已完整实现10种军种、10×10克制表、标签系统和配置表，通过DamageCalculator在战斗中实际应用克制计算。武将6维属性（武力/智力/统帅/政治/经济/忠诚度）通过属性映射影响兵种战斗表现，武将定位（统帅型/猛将型/谋士型/防御型）与兵种选择形成配合策略，详见 [Attribute属性系统设计.md](Attribute属性系统设计.md)。
