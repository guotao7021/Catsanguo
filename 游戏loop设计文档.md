完整游戏Loop设计文档（内政 → 战斗 → 奖励）
一、设计目标
🎯 核心目标

构建一个可持续循环、可商业化的游戏主循环（Core Loop）：

✅ 驱动玩家长期成长
✅ 建立资源 → 战斗 → 收益闭环
✅ 支持数值扩展与版本更新
✅ 兼容单机 / 联机 / 活动玩法
🎯 一句话定义

玩家通过内政积累资源 → 参与战斗消耗资源 → 获得奖励强化自身 → 继续发展

二、核心Loop结构
🔁 主循环（Core Loop）
内政建设 → 资源产出 → 战斗准备 → 战斗执行 → 奖励结算 → 成长强化 → 回到内政
🔁 子循环（子玩法）
短循环：单场战斗
中循环：关卡推进
长循环：城市发展 / 武将养成
